\doxysubsection{Detailed Description}
The base \mbox{\hyperlink{classGhoti_1_1Wave_1_1Server}{Server}} class. 

This class is the foundation of the Ghoti.\+io HTTP server. It serves as the interface to control and expand the server programmatically.

The constructor only creates the server object. It does not begin listening for connections. In order to begin listening for connections, the \mbox{\hyperlink{classGhoti_1_1Wave_1_1Server_af3cfbae96f3de9f6335fe8ce10928d23}{Server.\+start()}} function must be called.
